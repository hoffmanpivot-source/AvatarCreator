# AvatarCreator Roadmap

## Phase 1: Foundation (COMPLETE)
- [x] Vite + React + TypeScript project scaffold
- [x] R3F Canvas with OrbitControls
- [x] Template GLB loading + auto-centering
- [x] Morph target system (PARAM_KEYS, MORPH_SCALE, REMAP)
- [x] Skin tone application via material color

## Phase 2: Slider UI (COMPLETE)
- [x] 11 tabbed categories
- [x] 56 sliders (51 body params + 5 tune shape keys)
- [x] Left/right labels per slider
- [x] Double-click to reset slider
- [x] Tune sliders with different accent color

## Phase 3: Character Management (COMPLETE)
- [x] Save character to list (localStorage)
- [x] Load character from JSON file
- [x] Clone character
- [x] Export as JSON file (browser download)
- [x] Character list sidebar with select/delete
- [x] 4 built-in presets (Default, Athletic, Feminine, Curvy)
- [x] Keyboard shortcuts (Ctrl+S, Ctrl+N)

## Phase 4: Polish
- [ ] Loading spinner while GLB loads
- [ ] Preset tags editing
- [ ] Undo/redo
- [ ] Copy JSON to clipboard button
- [ ] Responsive layout for smaller screens
- [ ] Background color/gradient picker
- [ ] Grid toggle

## Phase 5: Advanced
- [ ] Side-by-side comparison (two models)
- [ ] Randomize button (random params within sane ranges)
- [ ] Import/export batch (multiple presets in one file)
- [ ] Screenshot capture (model only, no UI)
